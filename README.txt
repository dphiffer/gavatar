guild avatar generator
======================

adding a frame
--------------
1. copy the image (png, 1024x1024px) to `public/frames`
2. edit images.json
3. run `make dev` to test the changes locally
4. open a pr
5. for releases, see below

commands
--------
make dev           run local dev server at port 8000
make build         generate version.txt

environments
------------
- production
  https://propublicaguild.phiffer.org/

troubleshooting
---------------
- verify the sha-1 and the timestamp at https://propublicaguild.phiffer.org/version.txt
- verify that the correct images are included in https://propublicaguild.phiffer.org/images.json
